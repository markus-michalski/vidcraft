# Video Type: Social Short

## Overview

Ultra-short vertical videos optimized for social media feeds (TikTok, Instagram Reels, YouTube Shorts). The first 3 seconds determine whether viewers stay or scroll. Designed for muted autoplay with heavy text overlays, fast cuts, and a single punchy message that delivers value instantly.

## Characteristics

| **Aspect** | **Value** |
|------------|-----------|
| **Duration** | 15-60 seconds |
| **Pacing** | Very fast, scroll-stopping, relentless |
| **Tone** | Bold, casual, high-energy |
| **WPM** | 150-170 (rapid, punchy delivery) |
| **Scene Changes** | Every 3-10 seconds |
| **Primary Visual** | Avatar (vertical frame) + Bold Text Overlays + Quick Cuts |

## Structure Template

1. **Hook** (1-3s) — Stop the scroll: bold claim, question, or visual surprise
2. **Content** (10-45s) — Deliver the value: tip, insight, hot take, or mini-tutorial
3. **CTA** (3-8s) — Follow, comment, share, or link in bio

## Script Conventions

- Hook must work WITHOUT audio — text overlay is mandatory for the first 3 seconds
- Write for muted viewing: every key point must appear as on-screen text
- One idea per video — ruthlessly cut anything that doesn't serve the core message
- Use pattern interrupts: "Wait—", "Here's the thing—", "Stop doing this—"
- Speak in short, punchy phrases — fragments are fine
- Address the viewer directly: "You", "Your", never "One" or passive voice
- Max 10 words per sentence — shorter is better
- End with an open loop or engagement prompt: "Comment if you agree"

## Visual Direction

- **Format:** 9:16 vertical (1080x1920) — NEVER horizontal
- **Primary:** Avatar filling most of the frame, close-up or medium shot
- **Text Overlays:** Large, bold, high-contrast — readable on a phone at arm's length
- **Hook:** Full-screen text or dramatic visual change in frame 1
- **Transitions:** Jump cuts, zoom cuts, whip pans — no slow fades
- **On-Screen Text:** Every key point, stat, or quote as large overlay text
- **Captions:** Always — assume 80% of viewers have sound off
- **Safe Zones:** Keep text away from top 15% (platform UI) and bottom 20% (description/buttons)

## Pacing Rules

- Max 5 words of on-screen text at once (must be readable in 2 seconds)
- No scene longer than 10 seconds
- First 3 seconds are make-or-break — no intros, no logos, no "Hey guys"
- Every scene must have visual movement (zoom, cut, text animation)
- Use captions for 100% of narration
- If the video can't work on mute, it's not ready
- Total word count: 40-170 words (scales with duration)

## Anti-Patterns

- Starting with "Hey everyone" or any generic greeting
- Horizontal format or letterboxed content
- Small text that's unreadable on mobile
- Slow pacing or long scenes — viewers are gone in 2 seconds
- No captions or text overlays — most viewers watch on mute
- Multiple messages competing for attention
- Putting important text in platform-obscured zones (top/bottom)
- Ending without a CTA or engagement prompt

## Example Storyboard

```
Scene 1 (2s): Bold text overlay — "Your shop is losing sales because of THIS"
Scene 2 (4s): Avatar close-up + text — "90% of mobile users abandon checkout if it takes more than 3 seconds to load"
Scene 3 (6s): Avatar + quick screen recording — "Here's a 30-second fix: enable lazy loading for images"
Scene 4 (5s): Avatar + screencast — Show the one setting to toggle, highlight it
Scene 5 (3s): Avatar + text overlay — "Follow for more shop speed tips. Link in bio for the full guide."
```
