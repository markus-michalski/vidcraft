# Video Type: Webinar Teaser

## Overview

Short, energetic promotional videos designed to drive registrations for upcoming webinars or live events. They create urgency by highlighting the topic's value, showcasing the speaker's credibility, and making the key takeaways irresistible. Every second must push toward the registration CTA.

## Characteristics

| **Aspect** | **Value** |
|------------|-----------|
| **Duration** | 30-60 seconds |
| **Pacing** | Energetic, urgent, momentum-building |
| **Tone** | Exciting, authoritative, time-sensitive |
| **WPM** | 150-160 (confident, fast but clear) |
| **Scene Changes** | Every 8-15 seconds |
| **Primary Visual** | Avatar + Event Graphics + Speaker Info |

## Structure Template

1. **Topic** (5-10s) — What the webinar is about, framed as a burning question or challenge
2. **Speaker** (5-10s) — Who's presenting, why they're credible (brief)
3. **Takeaways** (10-25s) — 2-3 specific things attendees will learn or gain
4. **CTA** (5-15s) — Registration link, date/time, urgency element (limited spots, deadline)

## Script Conventions

- Open with the audience's pain point or goal, not the event title
- Frame takeaways as outcomes: "You'll walk away knowing how to..." not "We'll cover..."
- Keep speaker bio to one sentence — focus on relevance, not resume
- Include date, time, and timezone explicitly
- Create urgency: "Only 100 spots available", "Registration closes Friday"
- Use future tense to build anticipation: "You'll discover...", "We'll show you..."
- Max 15 words per sentence for energetic delivery

## Visual Direction

- **Primary:** Avatar delivers with high energy, direct to camera
- **Topic:** Bold text overlay with the core question/challenge
- **Speaker:** Photo/avatar of speaker with name, title, one-line credential
- **Takeaways:** Numbered list appearing on screen as avatar narrates
- **CTA:** Full-screen registration details with date, time, URL
- **Transitions:** Fast cuts, slide-ins, kinetic typography for key phrases

## Pacing Rules

- Max 8 words of on-screen text at once
- No scene longer than 15 seconds
- Build energy throughout — start strong, end strongest
- Takeaways should feel like a rapid-fire preview, not a detailed outline
- Registration details must be on screen for at least 5 seconds
- Total word count: 75-160 words (scales with duration)

## Anti-Patterns

- Leading with the speaker bio instead of the topic value
- Vague takeaways ("learn best practices" — which ones?)
- Missing date, time, or timezone in the CTA
- No urgency element — nothing motivating immediate action
- Spending too long on any single section, killing momentum
- Treating it like a webinar summary instead of a teaser

## Example Storyboard

```
Scene 1 (8s): Avatar + Text overlay — "Struggling to keep your shop GDPR-compliant after every plugin update? You're not alone."
Scene 2 (7s): Speaker card — "Join data protection expert Dr. Anna Weber, who's audited over 200 eCommerce shops."
Scene 3 (15s): Avatar + Numbered list — "You'll learn: 1. The 3 compliance checks to run after every update. 2. How to automate consent logging. 3. The one mistake that triggers 90% of GDPR fines."
Scene 4 (10s): Full-screen CTA — "Live on April 15th, 11:00 AM CET. Only 150 spots. Register free at example.com/webinar"
```
