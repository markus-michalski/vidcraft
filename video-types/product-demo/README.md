# Video Type: Product Demo

## Overview

Showcase videos that demonstrate a product's features, benefits, and user experience. Focus on what the product does and why it matters, not how to set it up. Designed to inform and persuade potential users or buyers.

## Characteristics

| **Aspect** | **Value** |
|------------|-----------|
| **Duration** | 2-5 minutes |
| **Pacing** | Medium, engaging, benefit-focused |
| **Tone** | Enthusiastic but honest, professional |
| **WPM** | 140-160 |
| **Scene Changes** | Every 15-30 seconds |
| **Primary Visual** | Mixed (avatar + screencast + slides) |

## Structure Template

1. **Hook** (5-10s) — Problem statement or attention grabber
2. **Product Introduction** (10-15s) — What is it, who is it for
3. **Feature 1** (30-45s) — Show feature → explain benefit
4. **Feature 2** (30-45s) — Show feature → explain benefit
5. **Feature 3** (30-45s) — Show feature → explain benefit
6. **Use Case / Story** (20-30s) — Real-world scenario
7. **Pricing / Availability** (10-15s) — Optional
8. **CTA** (10-15s) — Clear next step

## Script Conventions

- Lead with benefits, not features: "Save 2 hours per week" not "Has batch processing"
- Use "you" language: "You can..." not "Users can..."
- One feature per scene, max
- Show, don't tell: narration accompanies live demo
- Include a relatable pain point in the hook
- Avoid technical jargon unless audience is technical

## Visual Direction

- **Primary:** Alternating avatar and screencast/product shots
- **Avatar:** Professional, friendly, gestures to emphasize points
- **Product Shots:** Full-screen product UI with smooth navigation
- **Transitions:** Smooth slides and fades (more polished than tutorials)
- **On-Screen Text:** Feature names, key statistics, benefit statements
- **Graphics:** Comparison charts, checkmark lists, before/after

## Pacing Rules

- Fast scene changes (15-30s per scene)
- Max 3-5 key features (don't overwhelm)
- On-screen text for every key benefit (7 words max)
- Build energy toward the CTA
- Total word count: 300-700 words

## Anti-Patterns

- Feature dumping (listing every feature)
- No clear value proposition in the hook
- Missing CTA at the end
- Too technical for the target audience
- Demonstrating edge cases instead of happy path
- Video longer than 5 minutes (loses viewer attention)

## Example Storyboard

```
Scene 1 (8s): Avatar — "Managing product images in your shop shouldn't take all day."
Scene 2 (12s): Slides — Product name + tagline + key visual
Scene 3 (35s): Screencast — Drag-and-drop gallery upload, show instant preview
Scene 4 (30s): Screencast — Auto-optimization: before/after image quality comparison
Scene 5 (30s): Screencast — Responsive gallery on mobile device preview
Scene 6 (20s): Avatar — "One shop owner reduced their image workflow from 3 hours to 15 minutes."
Scene 7 (10s): Slides — Pricing + "Try free for 30 days"
Scene 8 (10s): Avatar — "Get started today — link in the description."
```
