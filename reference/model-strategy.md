# VidCraft Model Strategy

## Model Selection per Task

| Task Type | Model | Rationale |
|-----------|-------|-----------|
| **Creative writing** (scripts, briefs) | Opus | Needs creativity, nuance, natural language |
| **Reviews & validation** | Sonnet | Structured analysis, checklists, consistent |
| **Platform formatting** | Sonnet | Technical, rule-based, predictable |
| **Project management** | Sonnet | State reading, routing, status display |
| **Quick lookups** | Haiku | Fast, simple, informational |
| **Research** | Opus | Complex synthesis, source analysis |
| **Type creation** | Opus | Creative definition, comprehensive docs |
