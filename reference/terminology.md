# VidCraft Terminology

## Project Hierarchy

| Term | Definition |
|------|-----------|
| **Project** | A collection of related videos (e.g., "OXID Tutorial Series") |
| **Episode** | A single video within a project |
| **Scene** | A segment within an episode (one visual setup, one narration block) |
| **Shot** | A specific camera/visual configuration within a scene |

## Content Types

| Term | Definition |
|------|-----------|
| **Narration** | Spoken text delivered by the AI avatar |
| **On-Screen Text** | Text overlays visible to the viewer (titles, labels, commands) |
| **Visual Direction** | Instructions for what the viewer sees (not spoken) |
| **Script** | Combined narration + on-screen text + visual cues for an episode |
| **Storyboard** | Scene-by-scene visual blueprint with timing and transitions |
| **Brief** | Creative brief defining goals, audience, and constraints |

## Visual Types

| Term | Definition |
|------|-----------|
| **Avatar** | AI-generated presenter speaking to camera |
| **Screencast** | Screen recording of software/app |
| **Slides** | Presentation slides with text and graphics |
| **B-Roll** | Supporting footage or animations |
| **Split** | Split-screen layout (e.g., avatar + screencast) |

## Workflow Phases

| Phase | Description |
|-------|------------|
| **Concept** | Initial idea, goals, audience definition |
| **Brief** | Formal creative brief with constraints |
| **Research** | Source material gathering and analysis |
| **Script** | Writing narration and visual cues |
| **Storyboard** | Visual planning scene by scene |
| **Assets** | Screenshots, images, recordings collection |
| **Generation** | Creating video in HeyGen/Synthesia |
| **Review** | Quality assurance of generated video |
| **Publishing** | Distribution to target platforms |

## Platforms

| Platform | Key Characteristics |
|----------|-------------------|
| **HeyGen** | Custom backgrounds, gestures, screen share overlay, API access |
| **Synthesia** | Slide-based, 130+ languages, screen recording overlay, templates |
