# Synthesia Avatar Selection Guide

## Avatar Types

### Express Avatars
- AI-generated, available immediately
- 150+ options across demographics
- Support 130+ languages natively
- Best for: standard content, quick turnaround

### Studio Avatars
- Recorded from real actors
- More natural movement and expressions
- Limited language support (depends on actor)
- Best for: premium content, brand ambassadors

### Custom Avatars (Enterprise)
- Created from your own recording session
- Requires Synthesia studio session
- Most authentic brand representation
- Best for: CEO messages, company-specific content

## Slide Layout Options

| Layout | Avatar Position | Content Area | Use Case |
|--------|----------------|-------------|----------|
| **Full avatar** | Center | None | Intro, CTA |
| **Avatar + text** | Left 40% | Right 60% text | Explanations |
| **Avatar + media** | Left 30% | Right 70% media | Demos |
| **Overlay** | Bottom-right small | Full background media | Screencasts |
| **No avatar** | Hidden | Full screen | Text animations |

## Multi-Language Workflow

Synthesia's strength is multi-language production:

1. Write master script in primary language
2. Create video with primary avatar + language
3. Use Synthesia's translate feature for additional languages
4. Same avatar speaks all languages (Express avatars)
5. Review auto-translations for technical terms

## Voice Settings

| Setting | Range | Default | Notes |
|---------|-------|---------|-------|
| Speed | 0.5x - 2.0x | 1.0x | 0.9x for complex content |
| Pitch | -50 to +50 | 0 | Rarely needed |
| Pause | SSML tags | Auto | Add `<break time="1s"/>` for manual pauses |

## Background Options

- **Solid color:** Best for professional look
- **Image:** Upload custom background image
- **Video:** Loop a background video
- **Template:** Use Synthesia's built-in templates
- **Screen recording:** Overlay avatar on recorded content
