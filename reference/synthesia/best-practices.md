# Synthesia Best Practices

## Script Optimization

- Keep each slide under 1000 characters
- One idea per slide
- Use SSML-like punctuation for pacing: periods for pauses, commas for brief pauses
- Spell out technical terms phonetically if mispronounced
- Write numbers as words for narration clarity

## Slide Design

- Use Synthesia's built-in templates for consistency
- Avatar position: left or right (not center) when using media
- Keep text overlays to 3-5 bullet points max per slide
- Use high-contrast colors for text overlays
- Maintain consistent font size across slides

## Avatar Selection

- Synthesia offers 150+ avatars in 130+ languages
- Choose a "diverse" set for multi-language content
- Use custom avatars for brand consistency
- Test avatar with target language before full production

## Screen Recording Integration

- Use Synthesia's screen recording overlay feature
- Avatar appears as small overlay in corner during screencast
- Position avatar overlay in bottom-right for LTR languages
- Ensure screen recording resolution is at least 1080p

## Multi-Language Production

- Write master script in primary language
- Use Synthesia's translation features for localization
- Review auto-translations, especially for technical terms
- Maintain same avatar across language versions for brand consistency

## Common Issues

- Slide transitions: use simple cuts or fades, fancy transitions distract
- Text animations: use "appear" not "fly in" for professional content
- Long slides (>60s) lose viewer attention → split into multiple slides
- Background music: keep at low volume, consistent across slides
